# Ikimina Savings Rules

## Membership
- Groups have 5 to 500 members
- Members must agree on rules before joining

## Contributions
- Monthly fixed contribution
- Minimum: 10 XLM
- Maximum: defined by group

## Payouts
- One member receives payout per cycle
- Order is predefined and transparent

## Exit Rules
- Members cannot withdraw mid-cycle
- Rules are enforced automatically
