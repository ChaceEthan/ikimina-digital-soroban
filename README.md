# Ikimina Digital

Ikimina Digital is a decentralized savings and rotating payout platform built on the Stellar network.

The project aims to digitize traditional group savings (ROSCA / Ikimina) using smart contracts to ensure transparency, trust, and security.

## Key Goals
- Enable group savings using XLM
- Remove fraud and manipulation
- Support groups from 5 to 500 members
- Use Soroban smart contracts
